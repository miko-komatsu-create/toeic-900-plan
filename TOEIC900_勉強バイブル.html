基本の構え
1
音声が流れる前に写真を「動詞目線」で分析する
写真を見たとき、真っ先に確認するのは「誰が（主語）」「何をしている（動詞）」「どこに何がある（状態）」の3点。この3点を5秒で把握する。選択肢は必ず写真の何かを描写しているが、1つだけが正しい。
2
「写真に存在しないもの」が出てきたら即排除
TOEIC Part1の引っかけの9割は「写真に写っていないもの・人・行動」を選択肢に混ぜてくること。写真に車がないのに「A car is parked」が出たら問答無用で×。この判断を0.5秒で行う。
3
進行形か状態かを聞き分ける
「is being + 過去分詞」（進行形：今まさに〜されている）と「has been + 過去分詞」（完了形：〜された状態にある）は別物。「道路が工事中」なのか「道路が舗装された状態」なのかを写真で確認する。
頻出引っかけパターン
⚠ TRAP PATTERN 1

似た音の単語：「wood（木）」と「would」、「beach（浜）」と「bench（ベンチ）」など。聞こえた音と一致する単語ではなく、写真の内容と一致するかで判断する。

⚠ TRAP PATTERN 2

部分的に正しい描写：「The man is wearing a hat」→帽子の男性は写っているが、帽子をかぶる動作をしているわけではない。動作と状態を混同させる罠に注意。

⚠ TRAP PATTERN 3

主語すり替え：写真に男女がいて「She is〜」が正しいところに「He is〜」が出てくる。性別・人数・主語の確認を怠らない。

  • シャドーイング機能：Part1の音声を流してAI採点。発音の確認も同時にできる
  • おすすめ問題でPart1を指定して毎日5問。「正解してもレベル2（瞬時に解ける）か確認」する

冒頭1語の消去法システム
1
疑問詞（5W1H）で始まったらYes/Noを全消去
「Who / What / When / Where / Why / How」で始まる質問にYes/Noで答えることは不可能。この判断だけで選択肢を2〜3個から1〜2個に絞れる。冒頭の疑問詞を聞き逃したら全てが崩れる。
2
「or」で聞かれたら Yes / No は消去
「Would you like coffee or tea?」→Yes/Noで答えるのは不自然。どちらかを選ぶ or どちらも不要と答えるのが自然な応答。Orが聞こえたら選択疑問文と判定。
3
質問文と「似た音」の選択肢は罠
「Where is the meeting?」→「Weather（天気）」という単語が入った選択肢は引っかけ。TOEICは意図的に質問文の単語に似た発音の単語を選択肢に混ぜる。音が似ていると感じたら疑ってかかる。
4
間接応答に慣れる（最重要・最難関）
「いつ会議が始まりますか？」→「Tom に確認してください」という間接的な答えが正解になることが多い。直接答えていない選択肢が正解であることは900点レベルで頻出。「直接答えていない＝不正解」という固定観念を捨てる。
聞き取れなかったときの戦略
# 冒頭を聞き逃した場合のサバイバル戦術 Step 1: 主語を確認（I / you / he / she / we / they） Step 2: 動詞のニュアンスを追う（依頼か？確認か？許可か？） Step 3: 選択肢から「一番自然な会話の流れ」を選ぶ # 3秒で答えが出なければ → 即マークして次へ。1問で集中を乱さない。
abceed活用法
  • 移動中はPart2の音声を流してシャドーイング。口を動かせないなら頭の中で「これは5W1Hか、Yes/No疑問か、or疑問か」を瞬時に判定する練習
  • おすすめ問題でPart2を指定。正解した問題も「なぜ他の選択肢が不正解か」を解説で確認する

先読みシステムの確立
BEFORE AUDIO
設問文だけを先読みする（選択肢は見ない）
「What is the conversation about?」「What will the man do next?」疑問詞と動詞だけを捉える。選択肢を全部読む必要はない。ストーリーの方向性だけを把握する。
DURING AUDIO
ストーリーを追いながら「設問のキーワード」が出たら反応する
先読みした設問の疑問詞に対応する情報が会話中に出てきたとき、そこが答えのゾーン。全部理解しようとすると脳が過負荷になる。答えに関係する部分だけ集中する。
AFTER AUDIO
会話終了後、8秒以内に3問マークして次の先読みへ
「会話が終わった → 即マーク → 次の設問3問を先読み」のリズムを体に覚え込ませる。このリズムが崩れると連鎖的に全問落とすことになる。
よく出る設問タイプ別の攻略
A
「話者の目的」「場所」「職業」を問う問題
会話の冒頭30秒に答えがある。「I'm calling about〜」「We need to〜」といった冒頭の発言で場所・職業・目的が分かる。冒頭を絶対に逃さない。
B
「次にすること」「依頼」を問う問題
会話の最後の発言に答えがある。「I'll〜」「Can you〜」「Let me〜」が聞こえたら集中する。前半の内容は関係ない。
C
図表・グラフ連動問題（最難関）
画面上の表や地図と音声を組み合わせる問題。先読み時に表の内容を把握しておくのが必須。「Which meeting room is available?」→表で空き会議室を先読み→音声で条件を聞く。
シャドーイングの正しいやり方（必須）

問題タイプ別の解き方
A
品詞問題（全体の約40%）→ 3秒で解く
選択肢4つが全て同じ語幹（例：manage / manager / management / managerial）→品詞問題。語尾のルール：-tion/-ment/-ness → 名詞、-ive/-able/-al → 形容詞、-ly → 副詞、-ate/-ize → 動詞。空欄の前後の品詞を確認して3秒で解答。文の意味を考える必要はない。
B
動詞問題（時制・態・数）→ 15秒で解く
選択肢が同じ動詞の変化形（is done / was done / will be done / has been done）→動詞問題。確認するのは：①主語が単数か複数か ②時間軸（since / already → 現在完了、by the time → 過去完了）③能動態か受動態か。この3点だけ見る。
C
語彙問題（意味で選ぶ）→ 20〜30秒
選択肢が全て別の単語（例：affect / effect / reflect / defect）→意味問題。文全体を読んで文脈から判断する。金フレで覚えた単語がそのまま出る。語彙問題の正答率は金フレ習熟度に直結する。
D
前置詞・接続詞問題 → 10〜15秒
前置詞は名詞の前・接続詞は文の前。空欄の後が名詞句なら前置詞、SVを含む節なら接続詞。意味で迷ったら「despite（にもかかわらず）/ although（〜だけれども）/ because（〜なので）/ while（〜する間 / 一方）」の使い分けを確認。
でる1000問の使い方
# でる1000問 × abceedの完全攻略サイクル 1問解く └ 正解 → 「なぜ他が間違いか」を即答できるか確認 └ 不正解 → 解説を3回読む。「なぜこの品詞か」を言語化する 1セット（10〜20問）終了後 └ 間違えた問題だけを即時復習 └ abceedの「間違い問題絞り込み」機能を使って弱点問題だけを翌日出題 # 目標：「見た瞬間に構造が見えて3秒で解ける」状態（レベル2）
abceed活用法
  • でる1000問を毎問、時間計測しながら解く：1問ごとの秒数がわかるので、自分がどのタイプに時間がかかっているか特定できる
  • おすすめ問題でPart5を選択。「30問連続で時間を計って解く」ことで本番のリズムを養う
  • 苦手タイプ（品詞 or 語彙 or 文法）をabceedのスコア予測で特定して集中攻略

絶対に守る解き順
MUST DO FIRST
設問を先読みする（本文より先）
本文を読む前に設問文だけを読む。「What is the purpose of 〜?」「What is suggested about 〜?」疑問詞と何について聞かれているかだけを頭に入れる。選択肢まで読むと時間を浪費する。
THEN READ
本文を「設問のアンテナ」を張りながら読む
設問に関係する情報が本文に出てきた瞬間にキャッチする読み方。全てを理解しようとしない。設問の答えになりそうな部分だけに集中する。知らない単語があっても止まらず読み進める。
MARK & MOVE
答えが見えた瞬間にマーク → 次へ
本文を最後まで読んでから解答するのは非効率。設問の答えが見えた時点で即マークして次の設問へ。「確認のために全文読む」のをやめることで時間が大幅に節約できる。
設問タイプ別攻略
A
目的・要旨を問う問題 → 冒頭に答えあり
「What is the purpose of this email?」→ 本文の最初の段落に必ず答えがある。冒頭を読んで即解答。
B
詳細情報を問う問題 → スキャニングで探す
「When is the deadline?」「How much does it cost?」→ 設問のキーワード（deadline / cost / price）を本文でスキャンして該当箇所だけ読む。全文精読は不要。
C
NOT問題・推測問題 → 全選択肢を照合する
「What is NOT mentioned?」「What is implied about 〜?」→ 唯一時間がかかるタイプ。4つの選択肢を本文で1つずつ照合。時間がなければ最後に回す。
D
文挿入問題（Part7固有）→ 接続詞・代名詞・論理の流れ
「In which of the positions marked [1],[2],[3],[4] does the following sentence best belong?」→ 挿入する文の「This / These / However / Therefore」などに注目して前後の論理を確認する。

全体を通して
7つのパートを貫く
共通の真実
先人22人の体験記から共通して浮かび上がった、パート問わず効く本質。
復習の「3段階習熟度」理論
L1
わかる（じっくり考えればわかる）
辞書を引けばわかる。ゆっくり読めばわかる。この状態の知識はTOEICでは使えない。本番の処理速度に対応できない。
L2
瞬時にわかる（見た瞬間・聞いた瞬間に意味が出る）←目標
これがTOEIC本番で戦える唯一の習熟度。「できた」で終わらず「瞬時にできるか」を常に自問する。単語も文法も、見た瞬間・聞いた瞬間に答えが出ないなら、まだレベル1。
L3
使える（自分でも使いこなせる）
アウトプットまでできる状態。TOEICの点数だけを目的にするならL2で十分。ただし、L3に近いほど900点超えが安定する。
「1冊を完璧に」の法則

「多くの参考書を中途半端にこなすより、質の高い問題集を1冊徹底的に使う方がはるかに効果的。公式問題集を100回解いたとしても、それは答えを暗記するためではない。TOEIC特有のリズム・パターン・引っかけを『脳ではなく体』に覚え込ませるためだ。」

— 複数の体験記に共通する知見より
得点設計の原則
1
リスニングで高得点を取り、リーディングの失点をカバーする
900点のためのスコア配分例：L470〜490 + R410〜430。リスニングは満点（495）に近づけやすい。Part7を最後の5問捨てる前提で設計することで、残りの問題をゆっくり解けるメリットが生まれる。
2
語彙力がすべての土台
金フレを6〜7周すると、単語が「見た瞬間に意味が出る」状態になり、全パートの処理速度が同時に上がる。語彙強化は単語テストのためではなく、リスニング・リーディング両方の速度を上げるためのもの。
3
シャドーイングはリスニングではなくリーディングも強化する
音声を聴いて意味が取れる単語・表現は、文字で見たときも速く処理できる。リスニングの強化はリーディングの速度向上にも直結する。シャドーイングを軽視しないこと。

⚡

リスニングで最も集中力が要求されるのはPart4の後半。「今日は集中力があるな」という日でもPart4の最後の2〜3セットで落とすのが典型的なパターン。対策：Part4に入ったら意識的に「先読みのリズム」に戻る。

リーディングで最も集中力が低下するのは最後のトリプルパッセージ。「塗り絵（適当にマーク）」が始まるのは110〜120分目が多い。対策：週1回の2時間通し演習で脳の持久力を養う。本番で初めて120分集中するのは不可能。
試験中の集中力管理システム
1
「3秒ルール」を全パートに適用する
3秒考えて答えが出なければ、即マークして次へ。 1問に時間をかけすぎることは、後続の3〜5問を犠牲にする行為。期待値として最悪の戦略。「3秒」で見切る訓練を模試から習慣化する。
2
リスニング中に「先読みのリズム」を崩さない
1問で集中が乱れると、その問題の時間に次の設問の先読みができなくなり、連鎖的に3問を落とす。「1問を諦める勇気」がスコアを守る。諦めた問題は適当にマークして即次の先読みへ。
3
リーディングの時計管理：15分ごとのチェックポイント
残り75分:Part5スタート(15分で完了目標)
残り60分:Part6スタート(8分で完了目標)
残り52分:Part7スタート
残り5分:残った問題を全部マーク
4
リスニングとリーディングの間（約1分）を「リセット」に使う
リスニング終了→リーディング開始の数十秒間、深呼吸3回・目を閉じて頭をリセットする。多くの人がこの時間に問題を先読みしてしまうが、脳のリフレッシュに使う方が後半の集中力が大幅に上がる。

試験前日・当日の集中力最大化
前日夜 → 22時就寝
睡眠は最後の強化アイテム。6〜8時間の睡眠で翌朝の集中力が最大化される。前日の詰め込みよりも1時間多く寝る方が点数が上がる可能性が高い。
当日朝 → 軽い運動
10〜15分のウォーキングで脳への血流が増加し、集中力が向上する。試験会場まで1駅分歩くだけでも効果がある。
当日朝食 → 軽めに
満腹状態では血糖値の急上昇→急降下で眠くなる。試験中眠くなるのは最悪のシナリオ。炭水化物少なめ、タンパク質多めの食事を推奨。
当日朝 → 新しい勉強NG
当日朝に新しい単語や問題に取り組むのは厳禁。「知らない問題を見てしまった不安」が集中力を下げる。知っていることを軽く確認するだけで十分。

目標設定の心理学
★
「950点」を目標にする
「900点を目指す」と、脳は900点ギリギリの勉強を選択する。「950点」を目標にすると、勉強の深さ・精度・集中力が強制的に引き上げられる。900点が「通過点」になることで、結果として900点に届く確率が上がる。目標は常に「取りたいスコアより50点上」に設定する。
「壁」との向き合い方

「860点から905点に上がったとき、拍子抜けした。860点と905点の本質的な差は英語力ではなく、情報処理速度だった。超えられない壁だと思っていたものは、準備なしで挑んでいたから壁に見えていただけで、200時間の準備後に挑むと必ずワンランク上がった。」

— No.4 スタサプブログ体験記より
1
スコアが上がらない理由は「準備が足りないから」であり「壁があるから」ではない
「800点の壁」「900点の壁」と言われるが、壁は存在しない。適切な準備の量と質が足りていないだけ。逆に言えば、正しい準備を積み上げれば必ずスコアは動く。壁を嘆く時間を準備に使う。
継続のための設計思想
2
「着手のハードルを最小化」する
脳は「始めること」が最も苦手。「今日は金フレを1語だけ見る」でいい。1語見始めたら必ず続く。「何がなんでも1時間」ではなく「何がなんでも着手する」が継続の鉄則。疲れた日は5問で終わっていい。0にしないことだけが絶対ルール。
3
「疲労度別」に勉強内容を切り替える
頭が冴えているとき → 模試・でる1000問難問・Part7精読
普通のとき → シャドーイング・金フレ・でる1000問基本問題
疲弊しているとき → 金フレを眺めるだけ・音声を流すだけ
脳の状態に逆らって高負荷をかけても定着しない。適切な負荷が最速への道。
4
スコアが上がらない日を織り込む
模試のスコアが停滞・下がる日は必ずある。これは「蓄積している証拠」。植物が花を咲かせる前に根を張るように、スコアに出ない期間に力は蓄えられている。停滞は崩壊ではなく準備。
教育実習期間の特別なマインドセット
「減らさない」だけを目標にする
教育実習中に「増やそう」と思わなくていい。第1集中週で積み上げたものを「腐らせない」だけで十分。金フレ毎日・シャドーイング移動中・でる1000問夜30問、この3つを守るだけ。
移動時間を「勉強時間」と数える
移動中のシャドーイングは立派な勉強。「机に座っていないから勉強していない」という思い込みを捨てる。1日2時間の移動勉強は、週14時間の確実な積み上げになる。
「疲れたから今日はゼロ」は禁止
どんなに疲れても「金フレを1分見る」だけはやる。脳は「やらなかった」という記憶を「勉強は辛いもの」と学習してしまう。1分でも「やった」という事実が翌日の継続につながる。
最終週の自分への手紙を今書く
「6月6日から6月12日、あなたは全力で戦う。そのための準備を今している」という事実を、今日の自分が知っている。教育実習で疲れた夜に思い出すこと。
