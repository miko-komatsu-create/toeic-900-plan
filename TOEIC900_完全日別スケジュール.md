## ② abceedプロプランの具体的な使い方

### 基本の4機能

| 機能 | 具体的な使い方 |
|------|--------------|
| **金フレ（習得モード）** | 「わかった/わからなかった」で振り分け → AIが弱点語を自動再出題。音声は必ず聴く |
| **でる1000問（問題演習）** | 一問ごとに解説が出る。間違えた問題にチェックを入れて後で絞り込み復習 |
| **おすすめの問題** | 毎日のウォーミングアップに30分。AIが今の自分に必要な問題だけ出してくれる |
| **シャドーイング機能** | 公式問題集の音声を流してシャドーイング。AI採点で自分の発音精度も確認できる |

### 移動中（1日2時間）の固定使い方
```
往路（1時間）：
  └ 金フレ音声モード → 今日進める語の音声を聴きながら頭の中で意味を確認
  └ 「聞いた瞬間に意味が出るか」を検証する作業

復路（1時間）：
  └ 公式問題集11・12の音声（abceedに無料収録）を流してシャドーイング
  └ 口を動かせない環境でも「頭の中で追いかける（口パク）」だけでも有効
```
